## Project Overview

This project is a beginner-friendly exploration of building a PC-connected VR headset from scratch. Its purpose is to document the design and learning process, providing a hands-on introduction to virtual reality hardware and software.  

The project will focus on experimenting with concepts such as stereoscopic displays, head tracking, and PC-based rendering, while documenting the choices, challenges, and lessons learned along the way.
